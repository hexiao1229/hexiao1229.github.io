去京都跨年

发表于 2017-02-03 更新于 2020-03-27 Disqus：

由于出版业的性质，其实各行各业都差不多，每到年底都是一年中最忙的时刻之一。不管是赶业绩还是催回款，越到年底好像工作的弦就绷的越紧，我也是一样。写了几年不疼不痒的年终总结，今年终于也要开始为自己的发稿字数和回款实洋抗争了，每年觉得时间过的最快的时候就是完成一年总结的时候。由于今年年底和春节离得又近，我手里也有几本着急要赶在年前出书的汇编，2016年的最后几天，几乎都奔波在各个办公室之间了，进行着效率不高又不可避免的沟通工作。2016年12月29日是我生日，下午单位组织去走公园，我也就提早收拾好办公桌准备迎接2017年了。

这趟跨年的行程大致这样：24日Lana带着我妈前往东京，30日我们在京都汇合，跨年夜在京都清水寺，1号前往大阪住到3号，2号前往奈良一日游，3号从名古屋机场回国。此行一是带我妈走马观花的看看日本，再有就是重回京都慢慢欣赏，Lana之前也没有去过关西。这里不谈国家和国民的差异，只谈旅途中的见闻和感受，希望不引起歧义。

几年前我来过京都，当时连走马观花都算不上的就离开了，这次行程特意将京都的时间留的充裕一些，希望能慢慢体会。新年对于日本来说是重大节日，很多私营的旅店都在新年期间歇业，包括东京的不少博物馆和景点，都早早关门谢客。但京都有很大不同，虽然跨年期间少了一丝丝大城市的热闹，没有烟花和霓虹灯，但全年都向游客和信徒敞开的寺院或神社的大门，能让我在假日期间仍能享受到不一样的平静。这次在京都算是体验了日本大车站里面的自助Locker，也在京都体验了日本的公交车。从京都站出来就能看见去各个景点的车次，每一个队伍最后都有一名工作人员告诉你这趟车队尾的位置，无人抱怨无人插队。公车车门一侧是气动悬挂，方便福祉乘客。司机每个起步、转向都会告知乘客，下车也和每位乘客道别，这种感受是美好的。在日本，各行各业的人都是在做好本职工作，整个社会是往积极良性的方向发展，根本不用担心制度和混乱。最基本的，人多就要排队，买票才能坐车，垃圾不能乱扔，这些最基本最底线的东西根本不需要考虑。扯远了，回到京都。这个城市不大，最有名的景点几乎都是寺庙和神社。作为历史上的首都，京都的地位对于日本来说无需多言，对于游客来说文化遗产又是极为丰富的。不论寺院还是神社，最大的感受就是平和，安静是最大的特点，除去旅行团最多的金阁寺，我们在京都光顾的寺院都是极为安静的。但神社对于本地人来说在新年有着别样的意味，1日我们在伏见稻荷大社前被壮观的人流震撼到，感觉全京都的人都集中于此，吓得我们赶紧离开了。出了寺院，京都的街道在跨年夜前夕一样极为安静，除了一些临街仍然开放的商店，剩下几乎都是紧闭的大门，偶见房主人在2层露台清理落叶或者走过窗前，一切都是新年前的平静。

我在京都最大的遗憾是上次路过知恩院而没能进入，这次也算是满足个小小愿望。但与期待不同，可能是之前没想到高台寺留下的深刻印象，或是对京都有些审美疲劳，也可能冬日的地板太过冰凉，知恩院并没有想象中的那么美好。我们选择在京都徒步而不是自行车，所以下午时分，天气渐凉，又赶上南禅寺闭门，所以走到哲学之路就已疲惫，剩下了南禅寺、银阁寺，还有Lana碎碎念的西芳寺。也许没有雪的冬日，并不十分适合京都的气质，人们在冬天怀念夏天的感觉，而在夏天又怀念冬天的感觉。但落叶、夕阳、枯枝，加上冰凉的地板，配上京都的寺院，这感受我认为有点过了，就是已经超出自我认知的一种极致，感受也不那么好了。也许盛夏的哲学之路和寺院里的日本庭院，在寺院的阴影下喝上一听冰镇可乐，可能比在冬日里吃着日本人过年的点心、喝着加热的柚子茶一点也不差呢。

31日晚，我们重新回到京都开始的第一站清水寺，我们上午过门而不入，等着晚上来这里跨年。临近11点清水寺重新开门，没想到全是日本的年轻人一起组团来寺庙跨年。原本以为只有老人会选择去寺庙撞钟，没想到绝大多数都是和我们差不多的年轻人。夜晚的清水寺可能比白天还有热闹，虽然都是第一次去清水寺，但能赶上全年为数不多的夜间开放也算十分特别了。我特意准备了个八爪鱼的小三脚架用高感非常不行的GR拍拍夜景，等着临近12点的撞钟仪式。只是人群中出来几位僧人，念一段经文后开始撞钟，在0点前就由排队的民众开始撞钟，中途看热闹的人群突然开始倒计时，日本语的倒计时后迎来2017年，各国语言喊着新年快乐，这也算是完成了我第一个有所准备的跨年，还有那么一点点仪式感。

在伏见稻荷大社前，我们没有加入排队蠕动的人群，选择提前乘坐近铁前往大阪。大阪是我上次去日本的第一站，当时留下了很好的印象。但这次去却感觉不怎么样了，脏兮兮的街道，突然激增的人群，疲惫的帮亲戚朋友代购药妆，初到大阪的感受只剩下累觉不爱了。第二天出发前往奈良，日本第一个正式意义的首都。我们也是去奈良看鹿，其次是东大寺和春日大社，不能免俗。Lana从见到鹿开始就生怕错过，还没到博物馆就开始150、150的买鹿饼，玩的不亦乐乎。殊不知后面东大寺附近山中的鹿可能更加饥饿，更加缺乏游客的关爱。在喂鹿的过程，能体会到不同鹿有不一样的性格，有些温顺，有些激进甚至霸道。有的鹿会点头索取，有的鹿只会一味的吃饼头也不抬。在弱肉强食的时代，更温顺的鹿可能会被霸道的鹿抢食，人们还从中有所偏向和抉择，鹿在博弈哪位游客会给食物，人类会博弈哪只鹿比较温顺可爱或者是可怜。奈良的鹿是奈良的标志，谋杀快门的利器，鹿饼产业链重要的一环，吸引世界游客的工具，人与自然和动物和谐的典范，虽然我个人认为数量有点过多了。路过每年举办若草祭的若草山，看见光秃秃的一大片，想象着在二楼客栈躺着看若草祭的场景，这意淫起来也是极好的。由于新年第一天的缘故，春日大社的人就像北京的庙会一样多，路边摆摊卖小食品的摊位也如出一辙，吃点尝尝鲜，看见望不尽尽头的汹涌人群，我们还是先行离开了。

最后一天逛完大阪城公园，在难波附近逛逛商场，准备搭乘新干线前往名古屋机场，没想到日本春运来了闷头一棍。之前图便宜买了非指定席，其实当天买指定席都可能买不到，没想到非指定席3节车厢排队那个长啊，而且20分钟一趟车都感觉遥遥无期，我们又着急赶飞机，还要到JR名古屋换成到名铁站，实在等不及只能像当地人一样站到车厢连接处，站两站到名古屋去。新干线那么贵，没想到赶上新年假期人还是那么多，实在超出我的预料。通过这次也学会一点，尤其带着长辈出来玩，舒适还是最重要，能不转机不转机，能选近的机场就选近的。由于我买了往返名古屋的票，从大阪需要拖着4个行李箱前往常滑市那边的机场，家人很吃不消，算是吃一堑长一智了。

这篇文章完成于2月3日，中国农历春节假期已过，2017年也已经进入第二个月份。回想起来，我的2016是无比丰富的，是我人生重要的一年，结了婚、朝了圣、搬入了新房，算是开启步入中年的人生第二阶段。2016年的两段旅行经历仿佛过眼烟云一下就消失殆尽，精神层面的满足可能也已经疲劳，这从旅行前的兴奋程度可以体会。我对外面的世界依旧怀有憧憬，世界那么大，我所看到的还只有那么少，这想法虽然自私，但确是我每年想要走出去的动力。2017年，最想带老爸去发达国家看看，虽然不能前往敏感地区，但还是希望让他走出国门，去看看万恶的资本主义世界。
